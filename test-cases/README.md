# Test Cases

В этом разделе размещаются тест-кейсы, составленные в рамках бета-тестирования продуктов ВКонтакте.

## Что включает раздел
- функциональные тест-кейсы;
- проверку пользовательских сценариев;
- позитивные и негативные проверки;
- воспроизведение найденных дефектов.

## Назначение
Тест-кейсы используются для документирования результатов тестирования и последующего анализа качества продуктов VK.

Файлы будут добавляться и обновляться по мере выполнения тестирования.
