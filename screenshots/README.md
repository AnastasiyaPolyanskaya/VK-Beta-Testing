# Screenshots

В этом разделе хранятся скриншоты, использованные для оформления баг-репортов, чек-листов и тест-кейсов по продуктам VK.

## Что содержится в папке
- скриншоты шагов воспроизведения багов;
- подтверждения ошибок (фактический результат);
- визуальные артефакты, относящиеся к маскам, мини-приложениям, VK Питомцы, VK Чекбэк, Баланс Инь-Ян, Барсик и др.;
- изображения UI для анализа.

## Правила размещения
- Каждая группа скриншотов имеет уникальное имя папки или префикс по продукту.
- Формат файлов: **.png / .jpg**.
- Если скриншот относится к баг-репорту — указывается номер или краткое название дефекта.
