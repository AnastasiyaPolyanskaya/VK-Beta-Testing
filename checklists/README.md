# Checklists

Раздел содержит чек-листы, составленные в рамках бета-тестирования продуктов ВКонтакте.

## Назначение чек-листов
- Быстрая проверка основных функций продукта
- Контроль стабильности приложения
- Проверка UX/UI элементов
- Выявление ошибок перед составлением баг-репортов
- Поддержка регрессионного тестирования

## Структура раздела
В папке будут размещаться:
- функциональные чек-листы;
- чек-листы по UI/UX;
- регрессионные чек-листы;
- чек-листы конкретных продуктов (VK Pets, VK Balance, VK Mini Apps и др.).

Файлы будут добавляться по мере выполнения тестирования.
