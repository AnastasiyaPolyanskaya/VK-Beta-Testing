# Bug Reports

Раздел содержит баг-репорты, найденные во время бета-тестирования мобильных продуктов ВКонтакте.

## Что включают баг-репорты
Каждый баг-репорт оформлен по структуре:
- **Название дефекта**
- **Окружение** (устройство, ОС, версия приложения)
- **Предусловия**
- **Шаги воспроизведения**
- **Ожидаемый результат**
- **Фактический результат**
- **Вложения** (скриншоты / видео / логи)

## Какие продукты тестировались
В баг-репортах представлены проблемы из следующих продуктов VK:
- VK Приложение (основной функционал)
- VK Питомцы
- VK Мини-приложения (Mini Apps)
- Баланс Инь-Ян
- VK Чекбэк
- VK Маски (AR-фильтры)
- Колесо дня
- Барсик Lo-Fi кот

## Структура раздела
Здесь будут опубликованы:
- отдельные баг-репорты по каждому продукту;
- отчёты о найденных дефектах;
- анализы повторяемости и критичности.

Файлы добавляются по мере обработки багов.
