## Тест-кейсы проекта VK-01 - LoFi Кот Барсик

### Описание
В данном разделе представлены тест-кейсы, подготовленные в рамках исследовательского тестирования VK Mini App **LoFi Кот Барсик**.

Тест-кейсы направлены на проверку:
- корректности отображения карточек скинов;
- навигации внутри раздела «Предметы»;
- возможности взаимодействия пользователя с элементами интерфейса.

### Тип тест-дизайна
- Исследовательское тестирование (Exploratory)

### Объект тестирования
- VK Mini App: LoFi Кот Барсик
- Раздел: «Предметы» → «Скины»

### Основные проверяемые сценарии
- Открытие раздела «Предметы».
- Переход в подраздел «Скины».
- Проверка отображения карточек скинов.
- Проверка возможности пролистывания карточек свайпом.
- Проверка корректности адаптации интерфейса под разрешение экрана.

### Связанные дефекты
По результатам выполнения тест-кейсов выявлены дефекты отображения интерфейса.  
Подробные баг-репорты находятся в директории **bug_report/**.

### Формат тест-кейсов
- Тест-кейсы оформлены в виде таблицы.
- Каждый тест-кейс содержит:
  - ID
  - Название
  - Предусловия
  - Шаги
  - Ожидаемый результат
  - Фактический результат
  - Статус
