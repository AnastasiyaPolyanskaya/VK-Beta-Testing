# Проект VK-01 — LoFi Кот Барсик: Баг с отображением скинов (iOS)

## Описание
Проведено исследовательское тестирование раздела **LoFi Кот Барсик** в VK Mini App.  
Основное внимание уделено корректности отображения карточек скинов и возможности навигации между ними.

## Окружение
- ОС: iOS 18.2.6  
- Устройство: iPhone 14 Pro Max  
- Платформа: VK / VK Mini Apps  
- Приложение: LoFi Кот Барсик  
- Тип тестирования: Exploratory (исследовательское)  
- Применимость: Mobile (iOS)

## Проверенные сценарии
- Открытие раздела **Предметы** в LoFi Кот Барсик.  
- Переход в подраздел **Скины**.  
- Проверка корректности отображения карточек скинов.  
- Проверка возможности пролистывания (свайпа) скинов.

## Найденные дефекты
- Карточки скинов отображаются частично — правая часть экрана обрезана.  
- Отсутствует возможность пролистывания скинов свайпом.

Формальные баг-репорты находятся в папке **bug_report/**.
