# VK-Beta-Testing

Репозиторий с материалами по бета-тестированию продуктов экосистемы VK.  
Здесь хранятся артефакты тестирования: баг-репорты, чек-листы, тест-кейсы и скриншоты по различным направлениям VK.

## Направления тестирования

- **VK Mini Apps:** LoFi Кот Барсик  
- **VK.com:** модуль «Питомцы»  
- **VK Шаги** 

## Структура репозитория

- `bug-reports/` — оформленные баг-репорты  
- `checklists/` — чек-листы для смоук- и регресс-тестирования  
- `test-cases/` — тест-кейсы и сценарии  
- `screenshots/` — скриншоты, подтверждающие результаты тестирования  

Каждый проект внутри папок разделён на отдельные файлы с понятными названиями.

## Цель репозитория

Показать практический опыт тестирования мобильных и веб-продуктов VK:  
поиск дефектов, оформление документации, анализ поведения интерфейсов и UX.

Репозиторий дополняется по мере выполнения новых тестовых задач.
